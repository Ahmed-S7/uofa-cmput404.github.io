# cmput404.github.io

## Getting Started

0. `git clone https://github.com/uofa-cmput404/cmput404.github.io.git && cd cmput404.github.io`
0. `virtualenv venv --python=python3`
0. `source venv/bin/activate`
0. `make html && make serve`
